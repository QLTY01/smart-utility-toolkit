# Smart Utility Toolkit

A Flutter mobile application that provides multiple everyday conversion tools in one app.

## Features

- Length Converter
- Temperature Converter
- Weight Converter
- Currency Converter

## Screens Included

- Home Screen
- Length Converter Screen
- Temperature Converter Screen
- Weight Converter Screen
- Currency Converter Screen

## Built With

- Flutter
- Dart

## How to Run

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
